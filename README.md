# Privacy-Preserving Aadhaar Verification using Zero-Knowledge Proofs

## Overview

This project uses OCR, Poseidon hash, Circom, and Groth16 zk-SNARKs to verify identity information without revealing Aadhaar details.

## Workflow

Image/PDF
↓
OCR / Text Extraction
↓
Extract Name
↓
Poseidon Hash
↓
Circom Circuit
↓
Witness Generation
↓
Proof Generation
↓
Verification

## Technologies Used

- Node.js
- Tesseract OCR
- Circom
- SnarkJS
- Poseidon Hash
